# Examples

This folder contains example conversion settings, not source vehicles.

The settings are useful as references for:

- Which parts were translated, mirrored, or structurally mirrored
- How per-part offsets are stored
- How batch variant selection is represented

Use them through `Import Config` after loading your own matching source vehicle zip.

Converted vehicle zips are not included because this repository is MIT licensed and cannot include BeamNG source vehicle files under that license.
